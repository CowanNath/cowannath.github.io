openpyxl操作excel

一、workbook操作

workbook： 工作簿，一个excel文件包含多个sheet。
加载一个已有的excel：
    excel = openpyxl.load_workbook("./test.xlsx")
创建一个新的excel：
    excel = openpyxl.Workbook()
最后都要save一下：
    excel.save(file_path)

二、sheet操作

workbook： 工作表，一个workbook有多个，表名识别，如“sheet1”,“sheet2”等。
增
    ws = wb.create_sheet("Mysheet") #插入到最后(default)
    ws = wb.create_sheet("Mysheet", 0) #插入到最开始的位置
    ws = wb.create_sheet(title="Pip")
改
    ws.title = "New Title" # 修改表名称
查
    激活工作表:
    ws = wb.active
    ws = wb["frequency"] 
    读取所有sheet名字
    wb.sheetnames

三、单元格

cell： 单元格，存储数据对象
增
    写入数据操作的三种方式
    ws.cell(row=2,column=5).value=99
    ws.cell(row=3,column=5,value=100)
    ws['A4'] = 4 
改 重写和增相同
查
    # 读取单元格数据
    ws.cell(row=row,column=col).value
    ws['D1'].value 

    ws.max_row     # 获取行数
    ws.max_column  # 获取列数

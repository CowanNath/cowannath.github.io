Markdown日常使用方法

date 2024-07-24
folder 技术相关
tag Markdown

根据自己的使用习惯，记录一些相对基础的使用方法以供日常使用以及查阅。

1.标题和引用

有时，使用不同级别的标题来构造文档很有用。 用 # 开始创建标题。 行中的多个 ## 表示较小的标题。
你可以使用 1个 ＃ 到 6 个 ###### 表示不同尺寸的标题。

举例：

# 这是一个一级级标题
## 这是一个二级标题
### 这是一个三级标题

显示样式：

这是一个一级级标题

这是一个二级标题

这是一个三级标题

3.引用文本

举例：

>这是一个引用的文本

6.链接

具体使用![]()的方式即可插入索要需要的图片，[]为图片信息，()中填写链接地址，可为本地图片也可以为网络链接。

举例：
![二次元图片](../images/girl_hat_jewelry_1167505_2560x1440.jpg)

显示样式：

8.列表

可通过在一行或多行文本前面加上 -、* 或 + 来创建一个无序列表

举例：

    - jerry
    + 小丽
    * tommy

显示样式：

  • jerry
  • 小丽
  • tommy

要对列表排序，请在每行前面添加一个编号。

举例：

    1. jerry
    2. 小丽
    3. tommy

显示样式：

  1. jerry
  2. 小丽
  3. tommy

9.表格

可以使用竖线 | 和连字符 - 创建表。 连字符用于创建每列的标题，而竖线用于分隔每列。 必须在表格前包含空白链接，以便其正确呈现。

    | First Header  | Second Header |
    | ------------- | ------------- |
    | Content Cell  | Content Cell  |
    | Content Cell  | Content Cell  |
First Header Second Header
Content Cell Content Cell
Content Cell Content Cell

10.段落

通过在文本行之间留一个空白行，可创建新段落。
多一些空白行可使用 <br>的方式实现。

11.使用表情符号

可以通过在文章中使用:EMOJICODE:（冒号后跟表情符号的名称）的方式添加emoji表情。
具体表情样式链接：emoji-cheat-sheet

例如：😄😅🤩

12.忽略Markdown格式

通过在 Markdown 字符前面输入 \，可指示 GitHub 忽略 Markdown 格式（或对其进行转义）。
